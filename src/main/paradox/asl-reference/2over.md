
# 2over

## Signature

`a b -- a b a b`

## Summary

Duplicates the top two items on the stack. This operation is equivalent to `:over,:over`.
See the @ref:[over](over.md) operator for more information.

## Example

| **Before** | **After** |
|------------|-----------|
|            | **b**     |
|            | **a**     |
| b          | b         |
| a          | a         |
