
# rot

## Signature

`a b * -- b * a`

## Summary

Rotate the stack so that the item at the bottom is now at the top. To rotate in the reverse
direction use the @ref:[-rot](-rot.md) operator.

## Example

| **Before** | **After** |
|------------|-----------|
| c          | **a**     |
| b          | c         |
| **a**      | b         |
