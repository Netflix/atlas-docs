
# over

## Signature

`a b -- a b a`

## Summary

Duplicates the second item on the stack and pushes the copy to the top. See also:
@ref:[2over](2over.md) and @ref:[dup](dup.md).

## Example

| **Before** | **After** |
|------------|-----------|
|            | **a**     |
| b          | b         |
| a          | a         |
