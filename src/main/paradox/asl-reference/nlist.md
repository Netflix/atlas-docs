# nlist

## Signature

`aN ... a0 N -- aN List(aN-1 ... a0)`

## Summary

Create a list with the top N items on the stack. See also: @ref:[list](list.md).

Since: 1.5.0

## Example

| **Before**  | **After**    |
|-------------|--------------|
| `2`         |              |
| `c`         |              |
| `b`         | `(,b,c,)`    |
| `a`         | `a`          |
