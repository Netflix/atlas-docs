
# pick

## Signature

`aN ... a0 N -- aN ... a0 aN`

## Summary

Pick an item in the stack and put a copy on the top. See also: @ref:[dup](dup.md).

Since: 1.5.0

## Example

| **Before** | **After** |
|------------|-----------|
| **2**      | **b**     |
| d          | d         |
| c          | c         |
| **b**      | b         |
| a          | a         |
