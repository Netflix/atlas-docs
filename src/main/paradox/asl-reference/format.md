
# format

## Signature

`pattern:String args:List -- str:String`

## Summary

Format a string using a printf [style pattern]

[style pattern]: https://docs.oracle.com/javase/8/docs/api/java/util/Formatter.html

## Example

| **Before**      | **After**    |
|-----------------|--------------|
| `(,42,)`        |              |
| `i-%08x`        | `i-0000002a` |
