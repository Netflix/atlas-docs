
# des-epic-viz

## Signature

```
TimeSeriesExpr

```
     
## Summary

Helper for configuring @ref:[DES](../asl/des.md) in a manner compatible with legacy
epic alerts. For more information see the @ref:[epic macros](../asl/des.md#epic-macros)
section of the DES page.
