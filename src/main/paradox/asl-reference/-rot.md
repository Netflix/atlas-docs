
# -rot

## Signature

`* a b -- b * a`

## Summary

Rotate the stack so that the item at the top is now at the bottom. To rotate in the reverse
direction use the @ref:[rot](rot.md) operator.

## Example

| **Before** | **After** |
|------------|-----------|
| **c**      | b         |
| b          | a         |
| a          | **c**     |
