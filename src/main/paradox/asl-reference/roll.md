
# roll

## Signature

`aN ... a0 N -- aN-1 ... a0 aN`

## Summary

Rotate an item in the stack and put it on the top.

Since: 1.5.0

## Example


| **Before** | **After** |
|------------|-----------|
| **2**      |           |
| d          | **b**     |
| c          | d         |
| **b**      | c         |
| a          | a         |
