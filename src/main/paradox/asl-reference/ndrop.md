
# ndrop

## Signature

`aN ... a0 N -- aN`

## Summary

Remove the top N items from the stack. See also: @ref:[drop](drop.md) and @ref:[clear](clear.md).

Since 1.5.0

## Example

| **Before** | **After** |
|------------|-----------|
| 2          |           |
| c          |           |
| b          |           |
| a          | a         |
