    stack

    Input Stack:
    TimeSeriesExpr
    		 ⇨ Output Stack:
    StyleExpr

    Change the line style to be stack. In this mode the line will be filled to the previous stacked line on the same axis.

    See the line style examples page for more information.

    Example:

    BeforeAfter
    name,sps,:eq,
(,nf.cluster,),:by

    name,sps,:eq,
(,nf.cluster,),:by,
:stack
    Previous sort
    Next vspan
