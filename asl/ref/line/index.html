    line

    Input Stack:
    TimeSeriesExpr
    		 ⇨ Output Stack:
    StyleExpr

    Change the line style to be line. This is the default mode and usually does not need to be set explicitly.

    See the line style examples page for more information.

    Example:

    Default
    name,sps,:eq,
:sum,
:line
